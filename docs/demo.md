# Demo

This page presents example visualizations generated with NemoVR-Viewer.

---

# 2D Tracking Visualization

The 2D visualization mode displays synchronized camera recordings together with:

- DeepLabCut markers
- trajectory overlays
- playback controls
- frame navigation
- real-time visualization

# 3D Trajectory Visualization

The 3D visualization mode displays reconstructed trajectories inside a 3D environment.

This mode is useful for:

- spatial trajectory analysis
- reconstruction validation
- multi-camera inspection
- dynamic movement visualization

The viewer interface provides:

- subject and trial selection
- playback controls
- visualization mode selection
- trajectory display options
- DLC visualization
- export settings
- frame slider navigation
