## Features

* Multi-camera synchronized playback
* DeepLabCut overlay visualization
* 2D and 3D tracking visualization
* Trajectory trails
* Playback speed control
* Video export
* Interactive GUI

## Project Structure

```text
Results/
└── EXPERIMENT_ID/
    └── SUBJECT_ID/
        ├── Trial_cam0.mp4
        ├── Trial_cam1.mp4
        ├── Trial_DLC2D.npy
        ├── Trial_DLC3D.npy
        └── ...
```

Tracking files are generated by the NemoVR tracking pipeline.
