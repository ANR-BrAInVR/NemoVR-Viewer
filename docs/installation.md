# Installation
This page explains how to install and launch the NemoVR-Viewer application step by step.

The goal is to help users who may not be familiar with Python or programming environments.

---

# 1. Install Miniconda

NemoVR-Viewer uses a Python environment managed with **Conda**.

Conda allows the project to install all required dependencies automatically and avoids conflicts with other Python installations on your computer.

We recommend installing **Miniconda**, which is lightweight and easy to use.

Download Miniconda here:

* Windows/Linux installer:

```text id="1lfw8f"
https://docs.conda.io/en/latest/miniconda.html
```

Install Miniconda using the default installation settings.

After installation, restart your terminal.

---

# 2. Open a terminal

You will now create the Python environment required for the viewer.

## On Windows

Open:

```text id="r13v6g"
Anaconda Prompt
```

or:

```text id="9tk8wf"
PowerShell
```

---

## On Linux

Open a standard terminal.

---

# 3. Clone the repository

Choose a location on your computer where you want to store the NemoVR-Viewer project files.

For example, you may want to place the project on your Desktop.

---

## Open a terminal in your chosen location

### Example on Windows

If you want to place the project on your Desktop, run:

```bash id="2p7z0f"
cd "C:\Users\YOUR_USERNAME\Desktop"
```

Replace:

```text id="2qkgrm"
YOUR_USERNAME
```

with your own Windows username.

Example:

```bash id="7w9n7u"
cd "C:\Users\johnsmith\Desktop"
```

---

## Example on Linux

```bash id="9h9mzw"
cd ~/Desktop
```

---

## Clone the repository

Run:

```bash id="w6n8jc"
git clone https://github.com/ANR-BrAInVR/NemoVR-Viewer.git
```

This downloads all project files to your computer.

A new folder named:

```text id="n95fyw"
NemoVR-Viewer
```

will automatically be created.

---

## Move into the project folder

Run:

```bash id="j6r1e9"
cd NemoVR-Viewer
```

Your terminal is now located inside the project directory.

# 4. Create the Conda environment (ONE TIME ONLY)

This step only needs to be done once after downloading the project.

The following command creates an isolated Python environment containing all required libraries.

Run:

```bash id="x6c6l9"
conda create --name DLC-live3 -c conda-forge python=3.12 numpy matplotlib pyqt opencv
```

During installation, Conda may ask:

```text id="4i7d7v"
Proceed ([y]/n)?
```

Type:

```text id="a2r3j9"
y
```

and press Enter.

The installation may take several minutes.

---

# 5. Activate the environment

Each time you want to use NemoVR-Viewer, you must activate the environment first.

Run:

```bash id="g1q9pq"
conda activate DLC-live3
```

Once activated, your terminal should display:

```text id="a4w2pc"
(DLC-live3)
```

at the beginning of the command line.

---

# 6. Configure the results directory

Before launching the viewer, you must configure the location of your tracking results.

At this stage, you are no longer working in the terminal.

Instead, open the project folder on your computer using the file explorer.

Locate the file:

```text id="8w1t1q"
Settings.txt
```

And open it by double-clicking on it.

The file can be edited with the default text editor installed on your computer.

---

Inside the file, locate the following line:

```text id="v7h68h"
resultsDir
```

Modify it to point to your results folder.

Example:

```text id="5tt9m7"
resultsDir    'C:/Users/Desktop/Results'
```

Use the full absolute path to your tracking results directory.

Save the file once the modification is complete.

# 7. Launch the viewer

Once the environment is activated and the settings are configured, launch the application with:

```bash id="r5azr8"
python Viewer.py
```

The graphical interface should now open.

---

After the initial setup, you do NOT need to reinstall everything.

For future sessions, the procedure is simply:


---

## Step 3

Activate the environment:

```bash id="v5v91t"

```

---

## Step 4

Launch the viewer:


---

# 9. Expected results structure

The viewer expects tracking-generated files organized as follows:

```text id="2g07dz"
Results/
└── EXPERIMENT_ID/
    └── SUBJECT_ID/
        ├── Trial_cam0.mp4
        ├── Trial_cam1.mp4
        ├── Trial_DLC2D.npy
        ├── Trial_DLC3D.npy
        └── ...
```

The `.npy` files are generated by the NemoVR tracking pipeline.

---

# 10. Troubleshooting

## Conda command not found

If the terminal says:

```text id="g2v9gc"
conda: command not found
```

restart your terminal after installing Miniconda.

---

## Viewer does not open

Make sure the environment is activated:

```text id="3jz2wq"
(DLC-live3)
```

should appear in the terminal.

---

## Missing result files

Verify that:

* `resultsDir` is correctly configured
* tracking files exist
* the folder structure matches the expected format
